# Compression and transfer formats

Data engineers should recognize common compression and archive formats:

- `.zip`
- `.tar`
- `.tar.gz`
- `.gz`
- `.bz2`
- `.xz`
- `.zst`
- `.snappy`
- `.lz4`

These are common for datasets, logs, backups and data-lake transfer workflows. This repository does not include compressed samples because generated archives should normally remain artifacts.
