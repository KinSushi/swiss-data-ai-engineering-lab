# DevOps and cloud files

This section documents the operational files a Data & AI engineer should understand.

- Dockerfile and Docker Compose for reproducible local execution.
- GitHub Actions, GitLab CI, Azure DevOps and Jenkinsfile for CI/CD literacy.
- Terraform/HCL for infrastructure as code.
- Kubernetes, Helm and Ansible for cloud-native and operations awareness.
- `.env.example` for local configuration without committing real secrets.
