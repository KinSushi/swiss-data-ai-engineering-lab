# MLOps artifacts

The MLOps section demonstrates the artifacts required to move from a notebook to a controlled model lifecycle.

| Artifact | Path | Purpose |
|---|---|---|
| MLflow model descriptor | `mlops/MLmodel` | Standard model packaging metadata |
| Model card | `mlops/model_card.md` | Model purpose, limits, metrics, risks |
| Data card | `mlops/data_card.md` | Dataset scope, lineage, caveats |
| Monitoring plan | `mlops/monitoring_plan.md` | Data quality, drift, latency, errors |
| Audit trail | `mlops/audit_trail_example.jsonl` | Prediction-event traceability |
| Drift notes | `mlops/drift_detection_notes.md` | Concept/data drift awareness |
