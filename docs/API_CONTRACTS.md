# API contracts

This repository includes examples of API contract formats used in Data & AI systems.

- `api/openapi.yaml`: REST API contract.
- `api/asyncapi.yaml`: event-driven contract.
- `api/transaction.proto`: typed message schema for gRPC or streaming.
- `api/graphql/schema.graphql`: GraphQL query schema awareness.
- `api/postman/collection.json`: API testing collection.
- `api/wsdl/legacy_service.wsdl`: SOAP/WSDL legacy awareness.
