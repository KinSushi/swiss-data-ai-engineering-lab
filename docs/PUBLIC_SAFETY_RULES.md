# Public safety rules

## Forbidden

- real banking, insurance, health, client or employer data
- private school material
- exam subjects or answers
- grades
- certificates with identifiers
- CVs and cover letters
- job trackers and salary strategy
- recruiter messages
- secrets, tokens, credentials and private keys
- live trading claims or production decisioning claims

## Allowed

- synthetic data
- educational code written for this repository
- sanitized screenshots and SVG diagrams
- model cards and data cards based on synthetic data
- infrastructure examples without real cloud identifiers
- `.example` files with clearly fake placeholder values

## Pre-push checklist

- [ ] no real personal data
- [ ] no real client data
- [ ] no private documents
- [ ] no credentials
- [ ] no screenshots with personal identifiers
- [ ] no overclaiming of professional mastery
